# Currency Converter Chatbot

A chatbot that converts currencies in real-time.

## Technologies Used
- Python
- Flask
- Dialogflow
- Ngrok
- HTML

## How to Run
1. pip install flask requests
2. python app.py
3. ngrok http 5000
4. Open index.html with Live Server
